Technologies Used

Basic HTML was used to stucture website


The main source of styling for this website is TailWind CSS


JavaScript was used for fetching data from the API's, creating div elements inside of the containers that contained weather data, updating the time and weather, etc.


Info used on all the weather containers was pulled from the free version of the Open Weather Map API.


This API retrieves NASA's astronomy image of the day and gives it a description.

Logic


Current

The Currrent page of this website was constructed using accurate measurements directly pulled from the Open Weather Map API. The background image changes based on the current weather. The NASA image is also directly pulled from the NASA API, as well as its description.


36 Hours

The 36-Hour page of this website was constructed using a for each loop to display the weather every 3 hours. The free version of the Open Weather API does not fetch hourly data, hence why its every 3 hours.


Daily

The Daily weather page of this website was a little bit more complex. The free version of the API does not provide daily data, it only provides data for every 3 hours, for the next 5 days. For the temperature, I took the average temperature of each day, and I did not include the actual forecast of each day, because I would have done that by storing all the weather during that day into an array, and picking the one that occured the most but that would have been inaccurate.
